### 7.2.2 Mod-n arithmetic

$r \equiv i \mod n$<br>
$->$<br>
$r = i + qn$<br>
$---$<br>
$s \equiv j \mod n$<br>
$->$<br>
$s = j + tn$<br>
$---$<br>
$->$<br>
$r + s = i + j + (q + t)n$
$->$<br>
$r + s \equiv i + j \mod n$
$---$<br>
$rs = ij + (qj + ti)n + qtn^{2}$
$->$<br>
$rs \equiv ij \mod n$
